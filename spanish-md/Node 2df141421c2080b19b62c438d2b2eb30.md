# Node

Los **nodos** son los **bloques fundamentales de Godot**: cada objeto visible o funcional en tu juego es un nodo que se organiza en un **árbol de escenas**. **Node** es la clase base de **todo** en Godot.

## **¿Qué es Node?**

Node
├── Es el "papá" de todos los nodos
├── NO tiene posición, sprite ni física
├── Gestiona: timers, escenas, señales, hijos
└── Perfecto para lógica "invisible"

## **Características Principales**

| Propiedad | Función |
| --- | --- |
| **get_children()** | Lista todos los nodos hijos |
| **add_child()** | Agrega nodo hijo |
| **get_parent()** | Nodo padre |
| **queue_free()** | Elimina nodo (seguro) |
