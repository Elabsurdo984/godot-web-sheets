# Match

El **`match`** es la versión mejorada del **`switch`** (normalmente en C#, C++ y Java) tradicional: evalúa un valor y ejecuta el bloque que **exactamente coincide**, más potente que múltiples **`if/elif`** para estados discretos.

## Sintaxis Basica

```python
var estado = "idle"
match estado:
    "idle":
        print("Personaje quieto")
    "running":
        print("Corriendo")
    "jumping":
        print("Saltando")
    _:
        print("Estado desconocido")  # Default (_ = wildcard)

```

## Multiples Valores y Tipos

```python
var accion = "ataque"
match accion:
    "ataque", "defensa":  # Múltiples en una línea
        print("Acción de combate")
    1, 2, 3:             # Números
        print("Nivel bajo")
    var x:                # Binding (captura valor)
        print("Cualquier string: ", x)

```

## Comparación: Match vs if:

```python
# ❌ LARGO: Múltiples if
if tipo == "espada": pass
elif tipo == "arco": pass
elif tipo == "magia": pass

# ✅ LIMPIO: Match
match tipo:
    "espada", "arco": print("Físico")
    "magia": print("Mágico")
    _: print("Desconocido")

```

## **Poder Avanzado (Arrays/Patrones)**

```python
match [vida, energia]:
    [0, _]: print("Muerto")           # Vida=0, energía cualquier
    [_, 0]: print("Sin energía")      # Energía=0
    [100, 100]: print("¡Perfecto!")   # Exacto
```
