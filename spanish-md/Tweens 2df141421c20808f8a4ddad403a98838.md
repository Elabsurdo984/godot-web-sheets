# Tweens

Son el sistema oficial para crear **animaciones suaves por código**, interpolando propiedades (posición, escala, color, rotación) entre valores iniciales y finales de forma automática y eficiente.

## **Creación Básica**

```python
func _ready():
    var tween = create_tween()  # Crea tween automático
    tween.tween_property($Sprite, "position", Vector2(500, 300), 2.0)
    # Objeto | Propiedad | Valor final | Duración (segundos)
```

## Métodos Principales

| Método | Uso | Ejemplo |
| --- | --- | --- |
| `tween_property()` | Anima propiedades de nodos | `position`, `modulate`, `scale` |
| `tween_method()` | Anima funciones propias | Contadores, barras de vida |
| `set_ease()` | Suavizado (in/out) | `EASE_IN`, `EASE_OUT`, `EASE_IN_OUT` |
| `set_trans()` | Curva de transición | `TRANS_LINEAR`, `TRANS_SINE`, `TRANS_BOUNCE` |

## Ejemplo Completo

```python
func animar_boton():
    var tween = create_tween()
    tween.set_trans(Tween.TRANS_BOUNCE)
    tween.set_ease(Tween.EASE_OUT)
    
    # Secuencia: escala → color → rotación
    tween.tween_property($Boton, "scale", Vector2(1.2, 1.2), 0.3)
    tween.parallel().tween_property($Boton, "modulate", Color.RED, 0.3)
    tween.tween_property($Boton, "rotation_degrees", 360, 1.0)
    
    # Callback al finalizar
    tween.tween_callback(func(): print("¡Animación terminada!"))
```

## Tipos de Secuencia

```python
# Una tras otra (secuencial)
tween.tween_property(...)  # 1
tween.tween_property(...)  # 2 (ESPERA la 1)

# Paralelo (simultáneo)  
var paralelo = tween.parallel()
paralelo.tween_property(...)  # A
paralelo.tween_property(...)  # B (AL MISMO TIEMPO)
```
