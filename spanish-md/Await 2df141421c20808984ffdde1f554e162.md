# Await

El **`await`** es una palabra clave que **pausa la ejecución de una función** hasta que ocurra un evento asíncrono (señal, timer, otra función), sin congelar el juego entero.

## Sintaxis Basica

```python
# Esperar señal
await mi_nodo.pressed  # Pausa hasta que se emita "pressed"

# Esperar timer
await get_tree().create_timer(2.0).timeout  # Pausa 2 segundos

# Esperar otra función await
await mi_funcion_asincrona()
```

## **Casos de Uso Comunes**

```python
# Secuencia de diálogos
func dialogo():
    print("NPC: Hola")
    await get_tree().create_timer(1.5).timeout
    print("NPC: ¿Cómo estás?")
    await mi_boton.pressed  # Espera clic
    print("¡Gracias!")
```
