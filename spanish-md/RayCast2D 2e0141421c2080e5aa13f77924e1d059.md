# RayCast2D

**RayCast2D** es un **"rayo láser invisible"** que lanza una **línea recta** desde su posición para **detectar el primer objeto** en esa dirección. Perfecto para visión enemiga, detectar suelo, límites de salto.

## Propiedades Clave:

| Propiedad | Función |
| --- | --- |
| **target_position** | Largo/dirección del rayo |
| **is_colliding()** | ¿Golpeó algo? |
| **get_collider()** | ¿Qué golpeó? |
| **force_raycast_update()** | Actualizar manualmente |

## Codigo para una vision enemiga:

```python
extends CharacterBody2D

@onready var rayo_visión = $RayCast2D

func _physics_process(delta):
    # Apuntar rayo hacia jugador
    rayo_visión.target_position = (jugador.global_position - global_position).normalized() * 200
    
    if rayo_visión.is_colliding():
        var objetivo = rayo_visión.get_collider()
        if objetivo.is_in_group("jugador"):
            print("¡Jugador visible! Perseguir")
    else:
        print("No veo nada")

```

## **Casos de uso perfectos**:

- ✅ Visión IA enemiga
- ✅ Detectar suelo/bordes
- ✅ Apuntar armas
- ✅ Sensores de distancia

**Regla**: **SI necesitas "ver" en línea recta = RayCast2D**. **Area2D** para zonas circulares.
