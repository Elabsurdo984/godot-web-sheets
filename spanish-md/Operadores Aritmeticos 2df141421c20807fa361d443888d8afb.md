# Operadores Aritmeticos

| Operador | Acción | Ejemplo |
| --- | --- | --- |
| `+` | Suma | `5 + 3 = 8` |
| `-` | Resta | `5 - 3 = 2` |
| `*` | Multiplicación | `5 * 3 = 15` |
| `/` | División | `6 / 2 = 3.0` |
| `%` | Módulo (resto) | `7 % 2 = 1` |
| `**` | Potencia | `2 ** 3 = 8` |
