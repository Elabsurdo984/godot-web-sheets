# ShapeCast2D

**ShapeCast2D** es la versión **"avanzada"** de RayCast2D: lanza una **forma completa** (círculo, rectángulo, cápsula) en vez de un rayo, detectando **TODAS** las colisiones en esa área. Perfecto para "barrer" obstáculos adelante.

## Código Completo

```python
extends CharacterBody2D

@onready var shape_cast = $ShapeCast2D

func _physics_process(delta):
    # Barrer adelante 100px
    shape_cast.target_position = Vector2(100, 20)
    
    # Obtener TODAS las colisiones
    var colisiones = shape_cast.get_collision_count()
    for i in colisiones:
        var collider = shape_cast.get_collider(i)
        var punto = shape_cast.get_collision_point(i)
        print("Colisión con: ", collider.name, " en: ", punto)
        
        if collider.is_in_group("enemigos"):
            print("¡Enemigos adelante! Disparar")

```

## Casos de uso Poderosos

1. **Detección múltiple enemigos** (barrido frontal)
2. **Chequeo plataforma completa** (¿cabe el salto?)
3. **Radar 360°** (rotar ShapeCast2D)
4. **Búsqueda de items** en área
5. **Validación de movimiento** (¿choca con algo?)

## Configurar Inspector

Shape → RectangleShape2D (20x10)
Target Position → (80, 0)  # 80px adelante
Collision Mask → Jugador + Mundo
Max Results → 10  # Máximo enemigos detectados

## **Cuándo usar**:

- ✅ **Barridos frontales** (disparos cónicos)
- ✅ **Búsqueda masiva** obstáculos
- ✅ **Validación espacio** (¿puedo moverme?)
- ❌ **Detección simple** (usa RayCast2D)

**Regla**: **RayCast2D = mira, ShapeCast2D = barre**.
