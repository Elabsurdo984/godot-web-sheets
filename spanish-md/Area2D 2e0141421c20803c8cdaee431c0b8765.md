# Area2D

**Area2D** es el nodo para **detección de áreas** (triggers): detecta cuando objetos **entran/salen** sin bloquear movimiento. Perfecto para power-ups, zonas mortales, puertas automáticas.

## Señales Automáticas

```
# Pestaña Nodos, Conecta en editor o código
area_entered(body)      # Alguien entró
area_exited(body)       # Alguien salió
area_shape_entered      # Shape específico
body_entered(body)      # Body2D entró
body_exited(body)
```

## Código Completo (PowerUp)

```python
extends Area2D

signal powerup_recogido(valor: int)

@export var valor_puntos: int = 100

func _ready():
    # Conectar propia señal
    area_entered.connect(_on_area_entered)
    # O conectar señales automáticas
    body_entered.connect(_on_body_entered)

func _on_area_entered(area):
    if area.is_in_group("jugador"):
        powerup_recogido.emit(valor_puntos)
        queue_free()

func _on_body_entered(body):
    if body.is_in_group("enemigos"):
        print("¡Enemigo detectado!")
```

## Casos de uso Clásicos

✅ Power-ups (salud, balas)
✅ Zonas mortales (lava, pinchos)
✅ Puertas/triggers
✅ Collectibles
✅ Spawn points
✅ Checkpoints
✅ Radar/detección

**Regla de oro**: **SI solo necesita detectar ENTRAR/SALIR = Area2D**. **NUNCA** usa CharacterBody2D para triggers.
