# Funciones

Una **función** en Godot (GDScript) es un bloque de código reutilizable que realiza una tarea específica y se puede llamar desde cualquier parte del script. Se define con **`func`** seguido del nombre, parámetros opcionales y código ejecutable.

## Sintaxis Basica:

```python
func nombre_funcion():
	# Codigo de la funcion
```
