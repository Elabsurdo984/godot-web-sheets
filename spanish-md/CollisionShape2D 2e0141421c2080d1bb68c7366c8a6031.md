# CollisionShape2D

**CollisionShape2D** es el **"fantasma invisible"** que define la **forma de colisión** para nodos físicos. **NO dibuja nada visible**, solo detecta choques

## Formas de Colisión mas utilizadas:

| Shape2D | Uso | Ventajas |
| --- | --- | --- |
| **RectangleShape2D** | Paredes, suelos | **Rápido, simple** |
| **CircleShape2D** | Balas, ruedas | **Redondo natural** |
| **CapsuleShape2D** | Personajes | **Mejor para plataformas** |
| **ConvexPolygonShape2D** | Objetos irregulares | **Preciso** |

## Código Dinamico

```python
extends CharacterBody2D

@onready var collision_shape = $CollisionShape2D
@export var tamano: float = 50.0

func _ready():
    # Cambiar tamaño en runtime
    var rect = RectangleShape2D.new()
    rect.size = Vector2(tamano, tamano)
    collision_shape.shape = rect
```

**Regla**: **TODO nodo físico necesita EXACTAMENTE UN CollisionShape2D hijo**. Sin él = **sin colisiones**. Godot da un error.
