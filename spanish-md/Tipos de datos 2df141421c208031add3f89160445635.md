# Tipos de datos

En godot hay 2 clases de tipos de datos:

## Tipos Basicos:

No contienen subelementos.

Declaraciones:

```python
# Integer (numero entero)
var numero: int = 16

# Float (numero con decimales)
var pi: float = 3.14

# String (Cadena de texto), siempre entre comillas
var saludo: String = "Hola"

# Booleano (Verdadero o Falso)
var es_mayor_de_edad: bool = false

# Null (Valor Inexistente)
var nada: null
```

## Tipos Compuestos:

Incluyen estructuras para datos coleccionables:

```python
# Vectores (2D, 3D)

# Vector 2d, Vector2(x, y), x: Ancho, y: Alto
var posicion = Vector2(10, 10)

# Vector 3d, Vector3(x, y, z) x: Ancho, y: Alto, z: Profundidad
var posicion = Vector3(10, 10, 10) 

# Array (Lista ordenada de Variants)
var frutas: Array = ["pera", "manzana"]

# Dictionary (Clave: valor)
var edad_personas: Dictionary = {
	"Manuel": 10,
	"Juan": 20
} # Siempre una coma al final de cada clave valor

# Objects (puedes reemplazar Object por cualquier nodo)
var mi_objeto: Object = Object.new()
mi_objeto.propiedad = "valor"
```
