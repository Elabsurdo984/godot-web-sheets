# Condicionales

Los **condicionales** en GDScript son estructuras (**`if`**, **`elif`**, **`else`**) que ejecutan código diferente según si una condición es verdadera (**`true`**) o falsa (**`false`**). Controlan el flujo del programa tomando decisiones basadas en variables o eventos.

```python
if condicion:
    # Se ejecuta SI es verdadera
    print("¡Vida suficiente!")

elif vida <= 0:
    # "Sino si" - solo si la anterior falla
    print("¡Game Over!")

else:
    # "Sino" - si nada anterior se cumplió
    print("¡Murió!")

```
