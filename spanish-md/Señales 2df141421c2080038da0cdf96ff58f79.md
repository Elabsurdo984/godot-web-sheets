# Señales

Las **señales (signals)** son un sistema de comunicación **desacoplado** entre nodos o scripts: un objeto emite una señal y otros se **conectan** para reaccionar cuando ocurre

### Sintaxis Básica:

```python
# 1. DECLARAR señal (en emisor)
signal puntos_ganados(cantidad: int)

# 2. EMITIR (disparar)
puntos_ganados.emit(100)

# 3. CONECTAR (receptor)
func _ready():
    mi_nodo.puntos_ganados.connect(_on_puntos_ganados)

# 4. CALLBACK (función que recibe)
func _on_puntos_ganados(cantidad: int):
    total_puntos += cantidad
    print("¡Ganaste ", cantidad, " puntos!")

```

## Conexión Rápida (Editor)

1. Ve a pestaña **"Node"** → **"Signals"**
2. Doble clic en señal → Selecciona nodo receptor → "Connect"
