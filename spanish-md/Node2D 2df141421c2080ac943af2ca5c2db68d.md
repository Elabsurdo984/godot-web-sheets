# Node2D

**Node2D** extiende **Node** agregando **funcionalidad 2D**: posición, rotación, escala y transformaciones. Es la base para **todos los objetos visuales y físicos en 2D**.

## Propiedades Esenciales

```
position: Vector2(0, 0)     # x, y en píxeles
rotation_degrees: float     # 0-360 grados
scale: Vector2(1, 1)        # Tamaño (1=normal)
z_index: int                # Orden de dibujo
modulate: Color             # Color/Tint
```

## Cuando Usar Node2D

```
✅ Sprites móviles
✅ Objetos decorativos
✅ Balas, proyectiles
✅ Objetos padres de sprites
✅ Partículas 2D
❌ UI (usa Control/CanvasLayer)
❌ Física (usa RigidBody2D/CharacterBody2D)
```
