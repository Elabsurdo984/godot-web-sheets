# StaticBody2D

**Nodo inmóvil** para **colisiones estáticas** (suelo, paredes, plataformas). **No se mueve** por sí solo, **ultra eficiente** para escenarios.

## Propiedades Importantes:

```
En Inspector:
- CollisionShape2D (hijo OBLIGATORIO)
- Layers/Mask (qué colisiona)
- PhysicsMaterial (fricción, rebote)
```

## Codigo Minimo:

```
extends StaticBody2D

func _ready():
    # Opcional: Cambiar forma dinámicamente
    $CollisionShape2D.shape.size = Vector2(100, 50)
```

## Casos de Uso:

```
✅ Suelo/plataformas
✅ Paredes/muros
✅ Decoración sólida
✅ Límites de pantalla
❌ Objetos móviles (usa RigidBody2D)
```
