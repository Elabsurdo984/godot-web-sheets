# Array

Es una lista ordenada y redimensionable de elementos (números, strings, nodos, etc.) accesibles por **índice numérico** (empieza en 0). Perfecto para inventarios, enemigos, power-ups o secuencias de animaciones.

## Creación y Acceso

```python
var enemigos: Array[Node2D] = []  # Array tipado vacío
var powerups = ["salud", "velocidad", 100]  # Mixto

# Agregar
enemigos.append($Enemigo1)
powerups.push_back("escudo")

# Acceder (índice 0 = primero)
print(powerups[0])  # "salud"
print(powerups.size())  # 4 elementos
```

## Operadores Comunes

| Método | Acción | Ejemplo |
| --- | --- | --- |
| `append()` | Agregar al final | `array.append(10)` |
| `pop_back()` | Quitar último | `var ultimo = array.pop_back()` |
| `erase()` | Eliminar por valor | `array.erase("item")` |
| `clear()` | Vaciar todo | `array.clear()` |
| `size()` | Longitud | `if array.size() > 0:` |
