# Input Map

## ¿Qué es el Input Map?

El Input Map es la tabla donde defines **acciones** (jump, move_left, attack) y les asignas teclas, botones de mando o clicks de ratón.

En vez de preguntar “¿se presionó la tecla A?”, preguntás “¿se activó la acción **`move_left`**?”, lo que hace el código más limpio y fácil de remapear.

---

## Cómo se configura

1. Abrir Project → Project Settings → pestaña **Input Map**.
2. En el campo de arriba, escribir el nombre de la acción (por ejemplo **`move_left`**) y pulsar **Add**.
3. En la lista, pulsar el botón **+** de la acción y presionar la tecla o botón que quieras asignar (por ejemplo **`A`** o la flecha izquierda).
4. Repetir para **`move_right`**, **`jump`**, **`dash`**, etc

## Singleton Input (Uso Básico)

```python
func _physics_process(delta: float) -> void:
    var dir := Vector2.ZERO

    if Input.is_action_pressed("move_right"):
        dir.x += 1
    if Input.is_action_pressed("move_left"):
        dir.x -= 1
    if Input.is_action_pressed("move_down"):
        dir.y += 1
    if Input.is_action_pressed("move_up"):
        dir.y -= 1

    velocity = dir.normalized() * speed
    move_and_slide()
```

- **`Input.is_action_pressed("nombre")`** devuelve **`true`** mientras la acción está mantenida.
- Esto sirve para movimiento continuo (personaje, nave, etc.).

## Uso de Ejes

```python
func _physics_process(delta: float) -> void:
    var x_axis := Input.get_axis("move_left", "move_right")
    var dir := Vector2(x_axis, 0.0)

    velocity = dir * speed
    move_and_slide()
```

- **`Input.get_axis(negativa, positiva)`** devuelve un valor entre -1 y 1 según qué acción se esté presionando.
- Esto simplifica mucho el código de movimiento y se adapta bien a teclado y mando.
