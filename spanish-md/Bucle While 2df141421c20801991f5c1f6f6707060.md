# Bucle While

Repite código **mientras una condición sea verdadera** (**`true`**), ideal para situaciones donde no conoces cuántas iteraciones necesitas de antemano, como menús o esperas.

## Sintaxis Básica

```python
var contador = 0
while contador < 5:  # Mientras sea verdadero
    print("Contador: ", contador)
    contador += 1  # ¡IMPORTANTE! Incrementa o el bucle se hara infinito

print("¡Terminado!")  # Se ejecuta al salir

```

## **Diferencia clave con For**

| Bucle | Cuándo usarlo | Ejemplo |
| --- | --- | --- |
| **For** | Cantidad conocida fija | Recorrer 10 enemigos |
| **While** | Condición dinámica | Esperar input del jugador |
